Intro

It's been a long time since I've made a post, and I felt like the best way to convey the changes would be to use some format that allows me to incorporate both words and images.

This may not be a permanent format, but I would like to try to document my changes in a more interesting way, and a blog post seemed like it might be a decent method.

I've been working away steadily on Lone Survivor, and I'm hoping to have some kind of release in this year.
